Ты — фронтенд-разработчик, работающий в Replit. Твоя задача — превратить текущий «мусорный» внешний вид таблицы «Учебный план» в аккуратную, читаемую и отзывчивую реализацию.  

1. **Структура и отступы**  
   - Оберни таблицу в `<div className="overflow-x-auto border rounded-md">`.  
   - Установи одинаковые горизонтальные паддинги: `<th>, <td> { @apply px-3 py-2 }`.  

2. **Голова таблицы (header)**  
   - Сделай фон трёхуровневого header-щика градиентным:  
     - Уровень курсов: `bg-gradient-to-r from-slate-800 to-slate-700 text-white`.  
     - Семестры: `bg-slate-700 text-white`.  
     - Типы часов: `bg-slate-600 text-white`.  
   - Уменьши текст в третьем уровне до `text-xs uppercase`, выровняй по центру.  
   - Добавь тонкие разделители между уровнями: `border-b border-slate-500`.  
   - Зафиксируй весь блок header:  
     ```jsx
     <thead className="sticky top-0 z-20">
       …
     </thead>
     ```

3. **Левая «Дисциплины» колонка**  
   - Сделай её `sticky left-0 z-10 bg-slate-900 text-white` и задай ширину `min-w-[220px]`.  
   - Применяй `box-shadow` справа: `shadow-[2px_0_4px_-2px_rgba(0,0,0,0.4)]`.

4. **Зебра и группировка**  
   - Чередуй фон строк:  
     ```css
     tr:nth-child(odd) { @apply bg-slate-800 }  
     tr:nth-child(even){ @apply bg-slate-900 }
     ```  
   - Для разделов/групп задавай более тёмный фон: `.row-type-section { @apply bg-slate-700 }`.

5. **Интерактивные элементы**  
   - Разворачивающая стрелка: увеличь до `w-5 h-5` и поменяй цвет на `text-slate-300 hover:text-white`.  
   - Drag-grab: иконку «⋮⋮» увеличь до `text-lg` и добавь `cursor-grab`.

6. **Редактируемые ячейки**  
   - При фокусе на `<input type="number">` внутри `td` подсвечивай:  
     ```css
     td:focus-within { @apply bg-slate-700 ring-1 ring-primary }
     input { @apply w-full bg-transparent text-right px-1 }
     ```

7. **Итого-строка**  
   - Добавь `<tfoot>` с одной строкой «Итого»; зафиксируй её:  
     ```jsx
     <tfoot className="sticky bottom-0 bg-slate-900 text-white z-10">…</tfoot>
     ```

8. **Скроллбар**  
   - Сделай кастомный тонкий скроллбар:  
     ```css
     .overflow-x-auto::-webkit-scrollbar { height: 6px; }  
     .overflow-x-auto::-webkit-scrollbar-thumb { @apply bg-slate-600 rounded-full }  
     ```

9. **Адаптивность**  
   - На tablet (<1024px) прячь третий уровень header-а:  
     ```jsx
     <th className="hidden lg:table-cell">Лб</th>
     ```  
   - Позволяй горизонтальный скролл без обрезания контента.

Проверь в Replit, что:  
- При ширине экрана 1280px таблица остаётся читабельной.  
- Заголовки не накладываются друг на друга.  
- Фиксированные шапка и колонка работают корректно.  
- «Итого» всегда видна внизу.
